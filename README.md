# Earth Science Atlas 2022

프리미엄 감성의 고등학교 지구과학 학습 웹앱 프로토타입입니다. 한국 2022 개정 교육과정을 기반으로 확장 가능한 앱 셸과 **판 구조론 MVP 모듈**을 제공합니다.

## 포함 기능
- Earth Dashboard 홈 화면
- 판 구조론 인터랙티브 카드(발산형/수렴형/보존형)
- 지구 내부 단면 인터랙션
- 오개념 클리닉 카드
- 자료 해석 훈련소(플레이스홀더)
- 5문항 미니 퀴즈(즉시 피드백 + 해설)

## 기술 스택
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## 실행 방법
```bash
npm install
npm run dev
```

## 품질 점검
```bash
npm run lint
npm run build
```
